# node-extract

## Description

Extract archives (7z, zip, bz2, gz,...) using system executables

## Usage

```js
const extract = require('@mborne/extract');

await extract('/path/to/archive.zip');
```

## Options

| Name          | Required? | Description  | Default |
| ------------- | :-------: | ------------ | :-----: |
| `archivePath` |    YES    | Archive path |   NA    |
